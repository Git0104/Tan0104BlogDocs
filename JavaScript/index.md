# JavaScript とは

そもそも **JavaScript** とは何かを説明しようと思います。
JavaScript はプログラミング言語のひとつで、Web開発において欠かせない基本の言語です。

### 🚀 JavaScript で出来ること
* **Webサイトの動き:** ボタンを押したら色が変わる、画像がスライドするなど。
* **Discord Bot:** 自動返信やサーバー管理をするBotを作れます。
* **サーバー開発:** Webサイトの裏側の仕組みも作れます。

::: tip 拡張子について
JavaScript のファイルは `.js` という拡張子を使います。
:::

**例：**
```javascript 
console.log("Hello, JavaScript!");
```
